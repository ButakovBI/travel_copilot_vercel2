# AI Travel Copilot — Т-Путешествия

Помощник по планированию поездок с ИИ для сервиса «Т-Путешествия» (Т-Банк). Подбирает рейсы, отели и поезда по запросу пользователя, учитывает контекст (регион, траты, прошлые поездки при входе в ЛК). Уточняет параметры до 3 сообщений, формирует структурированный запрос, ищет в БД и выдаёт 3 лучших варианта через модель (DeepSeek или Qwen).

## Требования

- Node.js 18+
- Ключ DeepSeek API (`DEEPSEEK_API_KEY`) или Qwen (`QWEN_API_KEY`)
- API ID SMS.ru (`SMS_RU_API_ID`) для отправки кодов входа

## Установка и запуск

```bash
npm install
cp .env.example .env
# В .env задать: DEEPSEEK_API_KEY (или QWEN_*), SMS_RU_API_ID, DATABASE_URL

npx prisma db push
npm run db:seed

npm run dev
```

Откройте [http://localhost:3001](http://localhost:3001).

## Авторизация

Вход по номеру телефона: на указанный номер отправляется СМС с кодом (через SMS.ru). Код проверяется на бэкенде и хранится в БД с ограничением по времени. После успешной проверки создаётся сессия, пользователь сохраняется в БД (телефон, при необходимости имя и регион).

## Функциональность

- **Главная**: стиль Т-Путешествий — логотип Т-Банка, навигация (Частным лицам, Бизнесу, Премиум, Еще), «Личный кабинет», хлебные крошки (Путешествия › Отели), блок «Отели с кэшбэком до 10%», категории (Авиа, Отели, Поезда, Туры, Автобусы, Экскурсии, Избранное, Заказы), форма поиска (куда, даты, гости, кнопка «Искать»).
- **AI Copilot**: до 3 раундов уточнений (бюджет, направления, даты, гости, транспорт, комфорт), затем формирование запроса → поиск по БД → выбор 3 лучших вариантов моделью → вывод карточек с факт-чеком (цены из БД).
- **Корзина и оплата**: добавление в корзину, оплата через платёжный шлюз Т-Банка, подтверждение бронирования.

## LLM и регион

Используется DeepSeek (по умолчанию) или Qwen (OpenAI-совместимый API). Регион пользователя хранится в БД и передаётся в контекст (учёт при подборе направлений и формулировках; по умолчанию Россия).

## Развёртывание (Docker)

```bash
docker build -t travel-copilot .
docker run -p 3001:3001 \
  -e DEEPSEEK_API_KEY=... \
  -e SMS_RU_API_ID=... \
  -e DATABASE_URL="file:/app/data/dev.db" \
  -v travel-data:/app/data \
  travel-copilot
```

## Структура

- `app/` — страницы и API (auth с реальной отправкой СМС и проверкой кода, user context, copilot chat, cart, payment).
- `lib/` — БД (Prisma), SMS (SMS.ru), LLM (DeepSeek/Qwen), контекст пользователя, поиск с факт-чеком, Copilot (извлечение параметров, уточнения, поиск, ранжирование топ-3).
- `prisma/` — схема (User с region, PhoneVerificationCode, рейсы/отели/поезда/визы), seed.

Цены и условия в ответах берутся только из бэкенда; модель не генерирует их в тексте.
